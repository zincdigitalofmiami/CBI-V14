# Code Reviews

All code reviews organized by status.

## Structure

- **`active/`** - Ongoing reviews
- **`old/`** - Completed reviews
- **`new/`** - Pending review

## Usage

1. New review → Create in `new/`
2. In progress → Move to `active/`
3. Completed → Move to `old/`

## Examples

- `old/ENHANCED_CALCULATOR_REVIEW.md`
- `old/EVENT_PREDICTIONS_CODE_REVIEW.md`
- `old/CALCULATOR_DRY_TEST.md`
