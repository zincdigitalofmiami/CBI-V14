# Dashboard Pages Documentation

Document each dashboard page, its components, and functionality.

## Structure

- **active/** - Pages currently being developed
- **old/** - Completed/deprecated page docs
- **new/** - Recently created pages needing docs

## Page Documentation Template

```markdown
# [Page Name]

**Route:** /path/to/page  
**Status:** Active | In Progress | Deprecated  
**Last Updated:** YYYY-MM-DD

## Purpose
What this page does

## Components Used
- Component 1
- Component 2

## Data Sources
- BigQuery tables
- API endpoints

## Features
- Feature 1
- Feature 2

## Screenshots
(optional)

## Notes
Any special considerations
```

## Examples

- Vegas Intelligence Page
- Market Dashboard
- Forecast Views (1W, 1M, 3M, 6M)
- Settings Page
