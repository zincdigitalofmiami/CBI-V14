# Legacy Documentation

---

## Structure

- `handoffs/` - Completed session handoffs, cleanup documentation, and historical consolidation records
- `status/` - Completed status reports, legacy reference files, and operational status docs

---

## What's Here

This folder contains documentation that is:
- ✅ **Completed** - Tasks that are done (DEPLOYMENT_COMPLETE, END_OF_SESSION, etc.)
- ✅ **Historical** - Old cleanup/consolidation documentation
- ✅ **Legacy** - Files that are no longer actively used

---

## Active Documentation

For current, active documentation, see:
- `docs/plans/` - Active plans and strategies
- `docs/handoffs/` - Current handoff documents
- `docs/reference/` - Active reference documentation
- `docs/status/` - Current status (if any)
