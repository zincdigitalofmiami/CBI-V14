# System Configuration & Build Files

All system-level configuration, build files, and maintenance scripts.

## Structure

- **`config/`** - Build configs, deployment files, makefiles
- **`docs/`** - Contributor guides, license, setup docs  
- **`scripts/`** - System maintenance and setup scripts

## Files

### config/
- `Makefile` - Build automation
- `cloudbuild.yaml` - Cloud Build configuration
- `.pre-commit-config.yaml` - Git pre-commit hooks
- Other deployment and build configs

### docs/
- `LICENSE` - Project license
- `CONTRIBUTING.md` - Contribution guidelines (if moved)
- Setup and installation guides

**Note:** Some files (README.md, .gitignore, cloudbuild.yaml) must stay in project root per tool requirements.
