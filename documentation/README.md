# Documentation

All project documentation organized by status.

## Structure

- **`active/`** - Current living documentation
  - `api-references/` - API documentation
  - `guides/` - How-to guides
  - `system-docs/` - System documentation
- **`old/`** - Historical/outdated documentation  
- **`new/`** - Recently added, needs categorization

## Usage

1. New docs → Create in `new/`
2. Reviewed & current → Move to appropriate `active/` subfolder
3. Outdated → Move to `old/`

## Categories

- **API References:** REST APIs, integration docs
- **Guides:** Setup, deployment, usage guides
- **System Docs:** Architecture, data flow, integrations
